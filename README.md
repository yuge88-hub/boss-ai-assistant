# BOSS AI Assistant

> BOSS直聘全流程 AI 求职助手 — Claude Code Skill

一句话：**你告诉它找什么工作，它帮你搜、帮你说、帮你发简历、帮你换微信，检测到面试自动准备面试。**

---

## 完整工作流

```
Phase 1: 发现          Phase 2: 筛选         Phase 3: 诊断
┌──────────┐         ┌──────────┐         ┌──────────────────┐
│ 搜索关键词 │────────→│ 拉聊天列表 │────────→│ 看按钮：哪些做过   │
│ 批量打招呼 │         │ 按岗位过滤 │         │ 看消息：判断意图    │
│ 验证发送   │         │ 忽略不相关 │         │ 分类 A/B/C        │
└──────────┘         └──────────┘         │ 面试→自动准备面试！ │
                                           └────────┬─────────┘
                                                    ↓
Phase 5: 报告          Phase 4.5: 监控       Phase 4: 执行
┌──────────┐         ┌──────────┐         ┌──────────────────┐
│ A/B/C 分类 │←────────│ 等5-10分钟 │←────────│ 先消息→再简历     │
│ 联系方式   │         │ 循环2轮    │         │ →最后换微信/电话  │
│ 下一步建议 │         │ 有新回复就回 │         │ 每个验证 sureFound │
└──────────┘         └──────────┘         └──────────────────┘
```

## 功能一览

| 功能 | 说明 | 验证标准 |
|------|------|----------|
| 搜索打招呼 | 按关键词搜职位，逐个点击"立即沟通" | `changed-to-continue` |
| 智能筛选 | 自动识别目标岗位，忽略网约车/家政/建筑 | 岗位关键词匹配 |
| 意图判断 | 分析对方消息：要简历？要微信？要电话？面试？ | 消息类型分类 |
| 自动回复 | 根据岗位和语境生成针对性回复 | `status: sent` |
| 发简历 | 检测按钮状态，没发过的自动发 | `sureFound: true` |
| 换微信/电话 | 对方要联系方式时自动交换 | `sureFound: true` |
| 🔥 面试准备 | 检测到面试→自动抓JD→生成完整准备文档 | 速读版+完整版 |
| 循环监控 | 操作完等5-10分钟，回头查新回复 | 2轮封顶 |
| 分类报告 | A/B/C 分类 + 联系方式汇总 + 下一步建议 | 写入 progress.md |

### 面试准备（自动触发）

当检测到面试邀请时，自动：
1. 通过 `opencli boss detail` 抓取完整 JD
2. 搜索公司信息（成立时间、经营范围、法人）
3. 生成面试准备文档：
   - **速读版**：面试前5分钟看
   - **完整版**：JD权重解码 + STAR-L五层追问 + 回答框架 + 反问策略 + 薪资话术
4. 输出故事填空模板

**不需要手动粘贴 JD ——全自动完成。**

## 快速开始

### 前置条件

- 已安装 Node.js
- 已安装 Chrome 浏览器
- 已有 BOSS直聘 账号（求职者端）

### 安装

```bash
# 1. 安装 opencli
npm i -g @jackwener/opencli

# 2. 克隆本仓库
git clone https://github.com/yuge88-hub/boss-ai-assistant.git

# 3. 一键安装（Skill + opencli 命令扩展）
cd boss-ai-assistant
bash install.sh

# 4. 在 Chrome 打开 zhipin.com 登录
```

### 使用

在 Claude Code 中直接说：

```
"帮我搜营养师岗位，北京，15个"
"看看有没有新消息要回复"
"给我出一份今天的求职报告"
```

## 对话分类标准

| 类型 | 特征 | AI 动作 |
|------|------|---------|
| 🔴 面试邀请 | "面试"+"时间"/"邀请" | 回复确认 + **触发面试准备** → A类 |
| 🟠 要手机号 | "电话"/"手机号" | 回复直接给号码 |
| 🟠 要微信 | "微信"/"加一下" | 回复 + 平台交换微信 |
| 🟡 真人提问 | 手动打的、有具体问题 | 针对性回复 + 发简历 |
| 🟡 自动回复 | "我有一定XX经验，不甘于…" | 回复 + 发简历 |
| 🟢 对方已同意 | 简历被接受 | 不回复 → B类 |
| ⚪ 语音/图片 | 不便处理 | 回复"请发文字" |

## 常见问题

**Q: 需要我自己写代码吗？**
A: 不需要。安装后直接用自然语言操作。

**Q: 页面一直报 stale page identity？**
A: 杀守护进程自动重试，skill 已内置处理逻辑。

**Q: 换微信后对方没收到？**
A: 平台交换是发起请求，对方需要通过才行。看 verbose 输出确认 `sureFound: true`。

**Q: 多个同名联系人？**
A: 用公司名辅助匹配，skill 已内置。

**Q: 支持哪些城市？**
A: 所有 BOSS直聘 支持的城市，北京/上海/广州/深圳/杭州/成都等。

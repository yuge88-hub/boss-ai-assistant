# BOSS Job AI

> BOSS直聘 AI 求职 + 面试助手 — Claude Code Skill

**两大核心能力：**

| 🏃 求职自动化 | 🎯 面试准备 |
|:---:|:---:|
| 搜职位 → 打招呼 → 筛选 → 自动回复 | 检测到面试 → 抓JD → 公司扫描 |
| 发简历 → 换微信 → 换电话 | STAR-L追问 → 反问策略 → 薪资话术 |
| 循环监控 → A/B/C报告 | 速读版5分钟看完 + 完整版步步拆解 |

**一句话：投简历到拿offer，AI 全程帮你。**

```

         ↓                            ↓
         └────────  A类报告  ─────────┘
```

## 功能一览

### 🏃 求职引擎（boss-ai-assistant）

| 功能 | 说明 | 验证 |
|------|------|:--:|
| 搜索打招呼 | 按关键词搜职位，逐个"立即沟通" | `changed-to-continue` |
| 智能筛选 | 保留目标岗位，忽略网约车/家政 | 关键词匹配 |
| 意图判断 | 对方要简历？要微信？要电话？面试？ | 消息分类 |
| 自动回复 | 根据岗位+语境生成针对性回复 | `sent` |
| 发简历 | 检测按钮，没发过的自动发 | `sureFound` |
| 换微信/电话 | 平台交换或直接给出号码 | `sureFound` |
| 循环监控 | 每5-10分钟回头查新回复，2轮封顶 | — |
| 分类报告 | A/B/C + 联系方式 + 下一步建议 | progress.md |

### 🎯 面试引擎（interview-prep）

| 步骤 | 内容 |
|:--:|------|
| 0 | 面试前5分钟热身 |
| 0.5 | 公司扫描（天眼查/企查查） |
| 1 | JD真假判断 |
| 2 | 隐性硬门槛追问（原话模板） |
| 3 | JD权重解码（哪条是核心） |
| 4 | STAR-L 五层追问预判 |
| 5 | 回答框架（STAR-L / PREP） |
| 6 | 面试官类型策略 |
| 7 | 提问即信号映射 |
| 8 | 关键信号 + 不会答三步走 |
| 9 | 反问5道题 |
| 10 | 面试后60秒复盘 |
| + | 故事填空模板 + 薪资话术 |

## 快速开始

```bash
# 1. 前置依赖
npm i -g @jackwener/opencli

# 2. 克隆
git clone https://github.com/yuge88-hub/boss-job-ai.git
cd boss-job-ai

# 3. 一键安装
bash install.sh

# 4. Chrome 登录 zhipin.com，然后在 Claude Code 中说：
#    "帮我搜营养师岗位，北京，15个"
#    "帮我准备明天乐善康健的面试"
```

## 对话分类

| 类型 | 特征 | 动作 |
|------|------|------|
| 🔴 面试 | "面试"+"时间" | 回复确认 + **自动准备面试** → A类 |
| 🟠 要电话 | "电话"/"手机号" | 回复给号码 |
| 🟠 要微信 | "微信"/"加一下" | 回复 + 平台交换 |
| 🟡 提问 | 真人手动消息 | 针对性回复 + 发简历 |
| 🟡 自动回复 | "有一定XX经验…" | 回复 + 发简历 |
| 🟢 已同意 | 简历被接受 | → B类 |
| ⚪ 语音/图片 | 不便处理 | 回复"请发文字" |

## 常见问题

**Q: 需要写代码吗？**
A: 不需要。安装后自然语言操作。

**Q: 怎么触发面试准备？**
A: 两种方式——(1) 求职引擎检测到面试自动触发 (2) 直接说"帮我准备XX公司的面试"。

**Q: 页面报 stale page identity？**
A: 已内置自动重试，最多3次。

**Q: 支持哪些城市？**
A: BOSS直聘所有城市。
